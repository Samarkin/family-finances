# Agent Instructions

All agents working on this project MUST read the documentation located in the `docs/` directory before performing any tasks or making any changes to the codebase.

Key documents to review:

- `docs/architecture.md`: Project architecture and design principles.
- `docs/tasks.md`: Current task list and project progress.

## Code Quality

After every logically complete change, you MUST run:
- `npm run lint`
- `npm run format`
